Mapeo de preguntas típicas:
Harina: Hay harina panadera y pastelera, si es panadera + hinchado util para queques y si es panadera - hinchado
Chocolate: Para preparaciones sencilla (chocoteja) es mejor un chocolate barato, si es una torta premiun un chocolate premiun (mas costoso)
Aceite: Pastelero o freir
Crema de leche: + % de grasa, + tiempo de montado en un dulce; existe crema vegetal (no origin animal, distinto sabor)
